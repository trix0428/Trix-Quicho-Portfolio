TechnoFest

About

A TechnoFest inspired website for the Web Jam Competition by 9th National TechnoFest 2022. It is a 5-day (May 17 - 21, 2022) web development competition that requires the use of HTML, CSS, JavaScipt and Bootstrap only for the creation of the website. Presentation is held during the last day of the competition. A&T short for AJ Candelaria and Trix Quicho (me), created this website to purposely inform viewers about TechnoFest.

Challenges

This 4-day period competition is really challenging since AJ and I are in different time zones so communication is quite hard yet we managed to create this website by having different platforms of communication.

As a UI/UX Designer, one of the challenges is locating the assets such as images, name and dates of the events considering that TechnoFest is being celebrated every year in Far Eastern University.

Lessons Learned

  • I learned that some text may not be visually appealing knowing that there are different screen sizes from web to mobile. I should be considering the compatibility of the font style to different types of devices.
