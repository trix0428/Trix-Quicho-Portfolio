SamQuicksal

About

SamQuicksal is a capstone project for IT Web and Mobile Application. Its primarily purpose is to create a web and mobile application system that allows user to locate nearby samgyeopsal restaurant, book for reservation, queue in line using the app, order menu dishes, do payment within the app, give rating and enjoy rewards and promotions. Please note that this is only the mobile app design of SamQuicksal.

Problem Statement

Most customers are unable to wait for long queue lines and are unable to order because of shyness or a lack of confidence when raising their hand upon calling for a waiter.

Objective

To help customers know their queue while not patiently waiting in line and also to help customers order their preferred dish by using the application.
